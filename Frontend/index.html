NextHome

"NextHome is a PG rental management system designed to simplify the process of finding and managing rental accommodations. It allows users to browse available PGs, view detailed property information, and manage bookings efficiently. The system also provides features for property owners to list and manage their properties. It is built using .NET and Spring Boot, ensuring a scalable and structured backend."

Artisano

"Artisano is a full-stack e-commerce platform designed to support and promote handmade products by connecting artisans directly with customers. The platform allows users to explore a wide variety of unique handcrafted items, view detailed product descriptions, and interact seamlessly with sellers. It also provides dedicated features for artisans to list their products, manage inventory, and handle orders efficiently. The application focuses on delivering a clean and user-friendly interface, ensuring a smooth browsing and purchasing experience. Built using React for the frontend and Node.js, Express, and MongoDB for the backend, Artisano follows a scalable architecture and demonstrates strong full-stack development capabilities."
